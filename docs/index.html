Política de Privacidade –Tem Vaga O aplicativo Tem Vaga respeita a privacidade dos seus usuários e está comprometido com a proteção dos dados coletados. 1. Coleta de informações O aplicativo pode coletar informações técnicas necessárias para o seu funcionamento, como identificadores gerados pelo próprio aplicativo (exemplo: UUID) e dados armazenados localmente no dispositivo. 2. Uso das informações As informações coletadas são utilizadas exclusivamente para garantir o correto funcionamento do aplicativo, melhorar a experiência do usuário e manter as funcionalidades oferecidas. 3. Compartilhamento de dados O Tem Vaga não compartilha, não vende e não transfere dados pessoais dos usuários para terceiros. 4. Armazenamento e segurança Os dados são armazenados de forma segura e utilizados apenas enquanto necessários para o funcionamento do aplicativo. 5. Exclusão de dados O usuário pode solicitar a exclusão dos dados relacionados ao uso do aplicativo, quando aplicável. 6. Alterações nesta política Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que o usuário revise este documento sempre que utilizar o aplicativo. 7. Contato Em caso de dúvidas sobre esta Política de Privacidade, entre em contato pelo e-mail: contato@temvaga.app
